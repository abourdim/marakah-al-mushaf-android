# معركة المصحف — Battle of the Quran

Interactive trilingual web app based on **"Ma'rakat al-Mushaf fi al-Alam al-Islami"** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 knowledge cards on Quran preservation, attacks on the Quran, and orientalist claims refuted
- Historical timeline of Quran defense across centuries
- Interactive quiz
- Narrator with karaoke, speed/pitch control, sleep timer
- Three themes (Leather, Night, Parchment)
- Three languages (Arabic, English, French)
- Dua panel and daily wisdom

## Tech

Pure HTML/CSS/JS — no frameworks, no build step. Works offline as a PWA.
